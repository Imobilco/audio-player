h1. Аудио-плеер для аймобилки

Что должен уметь:

* Воспроизведение файлов (HTML5, Flash как фоллбэк)
* Сохранение последнец позиции воспроизведения файла на сервере
* Ставить метки/закладки в произвольных местах файла, синхронизировать с сервером
* Поддержка плэйлистов
* -Разнообразные обои- поддержка шкурок (нэймспейс в CSS, указываем ссылку на объект с контролами)
